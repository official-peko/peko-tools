===============================================================================
BUNDLED NATIVE COMPONENTS
===============================================================================

These components are written in C/C++/Objective-C and are compiled into the
`peko` toolchain and/or into applications built with it. They are invisible to
`cargo`, so this section is maintained BY HAND. Add an entry here whenever
third-party native source or a prebuilt native library is vendored.

Scope note: BearSSL, webview, and the Android native app glue are linked into
APPLICATIONS BUILT WITH PEKO, not just into the toolchain. An application that
uses networking, a webview, or targets Android is redistributing them, and must
carry these notices.


-------------------------------------------------------------------------------
BearSSL -- https://bearssl.org/
-------------------------------------------------------------------------------

The TLS implementation behind std::sockets. Vendored as prebuilt static
libraries for nine targets (toolkit/std/c/sockets/vendor/) plus headers
(toolkit/std/c/sockets/inc/). Unmodified.

Linked into every Peko application that opens a TLS connection.
