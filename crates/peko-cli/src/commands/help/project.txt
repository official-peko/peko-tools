peko project <SUBCOMMAND> [OPTIONS]

Creates, views, and modifies Peko projects.

SUBCOMMANDS
    new                   prompts for project info and creates a new
                          project folder in the current directory
    modify                prompts for new values for the current
                          project's metadata (name, bundle id, version,
                          target platforms)
    set-icon <PATH>       sets the project icon to the given image file
                          (1024x1024 recommended; PNG, JPG, JPEG, WebP,
                          ICO, or AVIF)
    show-info             prints the current project's configuration
    show-icon             opens a window that displays the project icon
    show-assets           opens the project assets folder in the system
                          file explorer
    add-asset <PATH>      adds an asset to the UI project
    remove-asset <NAME>   removes an asset from the UI project

OPTIONS
    --force               for `project new`/`project add-asset`, overwrite
                          an existing folder/asset with the same name

EXAMPLES
    peko project new
    peko project modify
    peko project set-icon ./icon.png
    peko project show-info
