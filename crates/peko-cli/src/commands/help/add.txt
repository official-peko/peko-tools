peko add [OPTIONS] <PACKAGE>

Adds a dependency to peko.toml and installs it.

Writes <PACKAGE> into the [dependencies] table, preserving the file's
formatting, then re-resolves and refreshes peko.lock. With no version flag
the requirement defaults to "*".

OPTIONS
    --version=<REQ>   pin a registry version requirement, e.g. ^1.2
    --path=<DIR>      add a local path dependency instead of a registry one

EXAMPLES
    peko add sockets
    peko add sockets --version=^1.2
    peko add local_dep --path=../local_dep
