peko pkg <SUBCOMMAND> [OPTIONS]

Authors and packs library packages for distribution.

SUBCOMMANDS
    new <NAME>            scaffold a new library at ./<NAME> containing a
                          peko.toml ([package] + [lib]), README.md, and a
                          source/lib.peko entry
    build                 pack the enclosing project into a .pkpkg
                          container next to the current directory. The
                          current directory must be inside a project
                          (one with a peko.toml).

OPTIONS
    --force               for `pkg new`, overwrite an existing folder
                          with the same name

EXAMPLES
    peko pkg new my_lib
    peko pkg build
