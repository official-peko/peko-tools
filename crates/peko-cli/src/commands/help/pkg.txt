peko pkg <SUBCOMMAND> [OPTIONS]

Authors and bundles host packages for distribution.

SUBCOMMANDS
    new <NAME>            scaffold a new package directory at ./<NAME>
                          containing Package.json, README.md, and an
                          initial v0.1.0 version folder
    build                 bundle the current package into a .pkpkg
                          file. The current directory must be inside
                          a package folder (one with a Package.json).

OPTIONS
    --force               for `pkg new`, overwrite an existing folder
                          with the same name
    --version=<VERSION>   for `pkg build`, bundle only the specified
                          version instead of every version

EXAMPLES
    peko pkg new my_lib
    peko pkg build
    peko pkg build --version=v1.2.0
