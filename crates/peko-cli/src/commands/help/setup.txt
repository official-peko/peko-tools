peko setup [OPTIONS]

Installs or updates the Peko development environment under ~/.Peko.

Downloads the Compiler SDK and the linux and android toolchains from the
public releases, provides the Apple SDKs through xcrun (on a macOS host),
installs the std and pekoui packages globally, lays in the toolchain
descriptors, records a versions manifest, and configures PATH. On a
machine that already has an install, it refreshes only the components
whose release has changed.

The Windows toolchain (MSVC CRT and Windows SDK) is optional and comes
through xwin. It is distributed under the Microsoft license, so it is
installed only when requested with --windows and the license is accepted
with --accept-microsoft-license.

OPTIONS
    --peko-version=<TAG>  install a specific peko CLI release (default:
                          the latest release)
    --sdk-version=<TAG>   install a specific SDK / toolchain release
                          (default: the latest release)
    --windows             also install the Windows toolchain via xwin
    --accept-microsoft-license
                          accept the Microsoft license for the Windows
                          toolchain (required with --windows)
    --force               reinstall every component even when its release
                          is already current
    --json                emit newline-delimited JSON progress events for
                          tooling (used by Peko Studio)

EXAMPLES
    peko setup
    peko setup --sdk-version=v2.0.0
    peko setup --windows --accept-microsoft-license
